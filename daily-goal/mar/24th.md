- 1. 다수의 카드 핸들링
     당장 본인의 안키만 해도 카드가 수천장 있는데 이를 변수 하나하나에 할당하면 유지보수 난이도가 말도 안된다.  
     따라서 필연적으로 동적 배열이 필요하다. // 동적 배열은 듣기만하고 아직 안배웠다. 2일차인데 벌써 새로운게 튀어나왔다.

- 2. ArrayList<WordCard>
```java
import model.WordCard;

import java.util.ArrayList;

ArrayList<WordCard> cards =  new ArrayList<>();

cards.add(new WordCard("단어", "뜻", "예문"));

for (int i = 0; i < cards.size(); i++) {
    cards.get(i).printCard();
        }
```
- `new ArrayList<>()`: 동적 배열 생성
- `add()`: 데이터 추가
- `get(i)`: i번째 요소 호출
- `size()`: 개수 확인
