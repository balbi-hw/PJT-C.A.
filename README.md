# Anki Clone

Anki 스타일의 반복 학습 카드 앱을 구현하는 프로젝트입니다.  
카드 생성, 조회, 복습 기능을 중심으로 학습용 서비스를 직접 개발합니다.

## Just out of curiosity
해당 프로젝트는 자바와 스프링 부트를 공부하던 중 강의만 듣는 **수동적 학습**에서 벗어나
배운 내용들을 직접 사용해보고 싶어서 기획하게 되었습니다.  
  
'본인이 사용할 프로그램을 처음부터 끝까지 혼자 완성해보는 경험' 은 누구에게 물어도 항상
중요하다고 하는 것이기에 이를 위한 프로젝트이기도 합니다.  
  
Anki 프로그램은 매일매일 사용하고 있으며 추가했으면 좋을 것 같은 기능도 있었기에
프로젝트 주제로 적합하다고 생각했습니다.
  
해당 프로젝트는 풀스택 개인 프로젝트로 진행합니다.

## Tech Stack
- Frontend: Node.js, React
- Backend: Java, Spring Boot
- Database: MySQL

## Features
- 카드 생성 / 수정 / 삭제
- 카드 복습
- 반복 학습 기반 복습 흐름
- 이미지 검색 연동 기능

## Goal
- Anki와 유사한 핵심 기능 구현
- 프론트엔드/백엔드/DB 연동 경험
- 실제 서비스 구조 학습
